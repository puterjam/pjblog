创作共用（Creative Commons）
-------------------------------------------------------------------------------
你可以免费: 拷贝、分发、呈现和表演当前作品；制作派生作品 ；
但是必须基于以下条款:

  署名。你必须明确标明作者的名字。.  
  非商业用途。 你不可将当前作品用于商业目的。 
  保持一致。 如果你基于当前作品更改、变换或构造新作品，你应当按照与当前协议完全相同的协议分发最终作品。 

对于任何二次使用或分发，你必须让其他人明确当前作品的授权条款 


升级（Update）
-------------------------------------------------------------------------------
 目录下有 update.rar 文件，里面包含两个版本的数据库升级文件。请用户看准blog的版本号进行升级数据库
   - update2.0to2.1.asp (测试版升级到正式版)
   - update2.1to2.2.asp （正式版 2.1 升级到 2.2）
   - update2.2to2.3.asp （正式版 2.2 升级到 2.3）
   - update2.3to2.4.asp （正式版 2.3 升级到 2.4）
   - update2.4to2.5.asp （正式版 2.4 升级到 2.5）

 把相应的升级文件上传到PJBlog2的根目录下执行即可
 针对 2.1 用户做了 专门的升级包可以到 http://www.pjhome.net/default.asp?id=397 下载
 针对 2.2 用户做了 专门的升级包可以到 http://www.pjhome.net/blogedit.asp?id=419 下载
 针对 2.3 用户做了 专门的升级包可以到 http://www.pjhome.net/article.asp?id=466 下载
 针对 2.4 用户做了 专门的升级包可以到 http://www.pjhome.net/article.asp?id=524 下载
 针对 2.5 用户做了 专门的升级包可以到 http://www.pjhome.net/article.asp?id=590 下载

 2.2 升级到 2.3 特殊说明
 为了确保升级成功，请先在2.2的后台执行 "数据库与附件"-"数据库管理"-"压缩修复"数据库操作

 2.4 升级到 2.5 修改了const.asp，手动升级的用户，记得重新修改此文件
 2.5 升级到 2.6 不用升级数据库，可以直接覆盖升级包文件接口
 2.6 升级到 2.7 可以不升级数据库,但是有个日志标题修复补丁,可以把标题字数增加到100个字

安装（Install）
-------------------------------------------------------------------------------
  1. 到PJBlog2官方论坛下载 "PJBlog2 27"
  2. 解压缩文件,然后把上传到服务器
  3. 修改const.asp里面的数据库路径和Cookie名称,以确保你的站点安全
  4. 用管理员用户登录
      用户名:admin 密码:admin888
  5. Enjoy!!!
