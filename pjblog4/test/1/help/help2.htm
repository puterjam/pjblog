UpLoadClass对象[专家说明]

１、可直接修改类设置自己需要的默认设置，修改行号分别为：
行70：m_Charset = "gb2312"　' 默认字符集
行71：m_TotalSize= 0　' 总上传大小限制，0为不限制
行72：m_MaxSize = 153600　' 单文件大小限制，0为不限制
行73：m_FileType = "jpg/gif"　' 文件格式限制，""为不限制
行74：m_SavePath = ""　' 文件相对保存位置，""为当前目录
行75：m_AutoSave = 0　' 保存模式，0为自动重命名保存

3、文件域不同时刻的状态比较

项目 对象
打开前 		对象打开后 文件保存后
无文件 有文件 无文件 有文件且符合 有文件不符合
.Form(Item) "" "" "" "" 目标文件名 ""
.Form(Item&"_TYPE") "" "" 文件类型 "" 文件类型 文件类型
.Form(Item&"_Name") "" "" 源文件名 "" 源文件名 源文件名
.Form(Item&"_Path") "" "" 上传路径 "" 上传路径 上传路径
.Form(Item&"_Size") "" "" 文件大小 "" 文件大小 文件大小
.Form(Item&"_Ext") "" "" 扩展名 "" 扩展名 扩展名
.Form(Item&"_Err") "" -1 "" -1 0 1/2/3

注：
1）""表示空字符串
2）.Form(Item)=""的时候，文件均不会保存到服务器上
3）自动保存模式（.AutoSave=0 or 1）文件保存与对象打开同时发生
4）手动保存模式（.AutoSave=2）文件保存发生在调用.Save(Item,FileName)时

4、同名文件域上传，仅处理第一个，其他忽略。
